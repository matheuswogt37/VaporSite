Tecnicamente o que é Vapor?

Peer-to-peer (P2P) é uma arquitetura de rede descentralizada na qual os participantes, chamados peers, interagem diretamente entre si sem a necessidade de uma autoridade central ou servidor. Cada peer atua como cliente e servidor, permitindo que compartilhem recursos e serviços diretamente com outros peers. Para começar a compartilhar recursos, o peer precisa criar um link magnético; esse link contém todas as informações necessárias para iniciar o download dos arquivos diretamente dos peers. Criar um link é inútil; para que outros peers participem, eles precisarão do link magnético.
O link magnético é normalmente compartilhado por sites como The Pirate Bay, 1337X, nyaa.si, YoutBittorrent, Limetorrents e outros. Todos eles apresentam sérios problemas, como dificuldade de busca, muitos anúncios, links falsos e descrições curtas.
Os dados de compartilhamento P2P nunca foram fáceis de criar e compartilhar com outras pessoas. A criação do link magnético é difícil e extensa, e a maioria das pessoas não sabe como fazê-lo. Para compartilhar este link magnético, existem muitas opções, mas nem todos acessarão o mesmo site. Há uma falta de centralização. Alguns sites possuem um mecanismo de busca, mas todos se concentram em apenas uma mídia.
Este projeto visa desenvolver uma API e um Cliente para uma aplicação desktop. A API visa centralizar todos os links, gerenciá-los e facilitar sua disponibilização para diversos dispositivos. O Cliente consumirá a API, mas trará muitas outras funcionalidades, como: integrações com o Qbit para criar e baixar arquivos; ajuda na busca avançada; e adição de alguns utilitários que podem eventualmente ajudar o usuário.
